Overview

XGBoost (eXtreme Gradient Boosting) is a highly optimized gradient boosting framework. It builds an ensemble of shallow trees sequentially, where each new tree corrects the residual errors of the current ensemble. Key innovations include second-order gradient approximation, built-in L1/L2 regularization on leaf weights, and efficient handling of sparse and missing data.

When to Use

  • Structured/tabular data with mixed feature types
  • Kaggle and ML competitions
  • When you need state-of-the-art predictive performance
  • Datasets with missing values

Pros

  • State-of-the-art accuracy on tabular data
  • Built-in regularization prevents overfitting
  • Handles missing values natively
  • Second-order optimization converges faster
  • Supports custom loss functions

Cons

  • Many hyperparameters to tune
  • Sequential training (harder to parallelize than RF)
  • Can overfit with too many rounds
  • Less interpretable than single trees
  • Sensitive to noisy labels

How It Works

  1. Initialize: Start with a constant prediction (e.g., the global mean).
  2. Compute gradients: For each sample, compute the first-order gradient $g_i$ and second-order gradient (Hessian) $h_i$ of the loss with respect to the current prediction.
  3. Fit a tree: Build a regression tree using a special split criterion based on $g_i$ and $h_i$, with regularization terms penalizing tree complexity.
  4. Update: Add the new tree's predictions (scaled by learning rate) to the cumulative ensemble prediction.
  5. Repeat for a fixed number of rounds or until convergence.

Key Equations

Additive Model

$$\hat{y}_i^{(t)} = \hat{y}_i^{(t-1)} + \eta \cdot f_t(x_i)$$

where $\eta$ is the learning rate and $f_t$ is the $t$-th tree.

Regularized Objective

$$\mathcal{L}^{(t)} = \sum_{i=1}^n \ell\!\left(y_i, \hat{y}_i^{(t-1)} + f_t(x_i)\right) + \Omega(f_t)$$

$$\Omega(f_t) = \gamma T + \frac{1}{2}\lambda \sum_{j=1}^T w_j^2$$

where $T$ is the number of leaves, $w_j$ are leaf weights, $\gamma$ penalizes tree complexity, and $\lambda$ is L2 regularization.

Second-Order Approximation

$$\mathcal{L}^{(t)} \approx \sum_{i=1}^n \left[g_i f_t(x_i) + \frac{1}{2}h_i f_t(x_i)^2\right] + \Omega(f_t)$$

where $g_i = \partial_{\hat{y}} \ell(y_i, \hat{y}_i^{(t-1)})$ and $h_i = \partial^2_{\hat{y}} \ell(y_i, \hat{y}_i^{(t-1)})$.

Optimal Leaf Weight

$$w_j^* = -\frac{\sum_{i \in I_j} g_i}{\sum_{i \in I_j} h_i + \lambda}$$

Split Gain

$$\text{Gain} = \frac{1}{2}\left[\frac{(\sum_{i \in I_L}g_i)^2}{\sum_{i \in I_L}h_i+\lambda} + \frac{(\sum_{i \in I_R}g_i)^2}{\sum_{i \in I_R}h_i+\lambda} - \frac{(\sum_{i \in I}g_i)^2}{\sum_{i \in I}h_i+\lambda}\right] - \gamma$$

Python
import xgboost as xgb
from sklearn.model_selection import train_test_split
from sklearn.metrics import mean_squared_error, r2_score

# Split data
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# Fit model
model = xgb.XGBRegressor(
    n_estimators=200,
    max_depth=4,
    learning_rate=0.1,
    reg_alpha=0.0,       # L1 regularization
    reg_lambda=1.0,      # L2 regularization
    subsample=0.8,
    colsample_bytree=0.8,
    tree_method='hist',  # fast histogram-based
    random_state=42
)
model.fit(X_train, y_train,
          eval_set=[(X_test, y_test)],
          verbose=False)

# Predict
y_pred = model.predict(X_test)

# Evaluate
print(f"R-squared: {r2_score(y_test, y_pred):.4f}")
print(f"RMSE:      {mean_squared_error(y_test, y_pred, squared=False):.4f}")

# Feature importance
xgb.plot_importance(model, importance_type='gain', max_num_features=10)
