Overview

CatBoost (Categorical Boosting) is a gradient boosting library developed by Yandex. It introduces ordered boosting to reduce prediction shift (a form of target leakage) and symmetric (oblivious) decision trees that use the same split condition at each level, enabling extremely fast inference via bitwise operations.

When to Use

  • Data has categorical features that are expensive to one-hot encode
  • You want strong out-of-the-box performance with minimal tuning
  • Fast inference is critical (production deployment)
  • You need GPU-accelerated training

Pros

  • Native categorical feature handling
  • Ordered boosting reduces overfitting
  • Symmetric trees enable very fast inference
  • Works well out-of-the-box with default parameters
  • GPU training support

Cons

  • Training can be slower than LightGBM
  • Symmetric trees may underfit complex patterns
  • Large model files due to oblivious tree storage
  • Fewer community resources than XGBoost

How It Works

Ordered Boosting

Standard gradient boosting computes residuals using the model trained on the same data, which causes prediction shift. CatBoost uses a permutation-based ordered boosting approach: for each sample, the model is only trained on preceding samples in a random permutation, eliminating target leakage.

Symmetric (Oblivious) Trees

At each depth level, all nodes use the same split feature and threshold. This means a tree of depth $d$ has exactly $2^d$ leaves. The symmetric structure acts as implicit regularization and allows prediction via simple bit operations.

Ordered Target Encoding

For categorical features, CatBoost computes target statistics using only preceding samples in the random permutation, preventing target leakage that plagues naive target encoding.

Key Equations

Ordered Target Statistic

$$\hat{x}_i^k = \frac{\sum_{j:\ \sigma(j)<\sigma(i)} [x_j^k = x_i^k] \cdot y_j + a \cdot P}{\sum_{j:\ \sigma(j)<\sigma(i)} [x_j^k = x_i^k] + a}$$

where $\sigma$ is the random permutation and $a, P$ are prior parameters.

Symmetric Tree Split

At depth $d$, all $2^{d-1}$ nodes at level $d$ share the same split: $$(j_d, t_d) = \arg\max_{j,t} \sum_{\text{nodes at level } d} \text{Gain}(\text{node}, j, t)$$

Prediction (Oblivious Tree)

$$f(x) = w_{\text{leaf}(x)}, \quad \text{leaf}(x) = \sum_{d=1}^{D} 2^{d-1} \cdot \mathbf{1}[x_{j_d} > t_d]$$

Python
from catboost import CatBoostRegressor
from sklearn.metrics import mean_squared_error, r2_score

model = CatBoostRegressor(
    iterations=500,
    learning_rate=0.1,
    depth=6,
    l2_leaf_reg=3.0,
    cat_features=cat_cols,  # list of categorical column indices
    verbose=50
)
model.fit(X_train, y_train, eval_set=(X_test, y_test), early_stopping_rounds=20)

y_pred = model.predict(X_test)
print(f"R-squared: {r2_score(y_test, y_pred):.4f}")
print(f"RMSE:      {mean_squared_error(y_test, y_pred, squared=False):.4f}")
