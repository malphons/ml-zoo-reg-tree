AdaBoost Regressor

Regression Tree Est. 1997

Sequentially fits weak learners on re-weighted data, focusing on samples with high residuals. The final prediction is a weighted combination of all weak learners.

Boosting rounds:3

Overview

AdaBoost.R2 adapts the AdaBoost algorithm for regression. It trains a sequence of weak regressors (typically shallow trees), where each subsequent learner focuses more on samples that the previous learners predicted poorly. The final prediction is a weighted median of all weak learners, where weights reflect each learner's accuracy.

When to Use

  • You want a simple, effective boosting baseline
  • Data has moderate noise (AdaBoost can overfit on noisy data)
  • You need interpretable weak learner weights

Pros

  • Simple and well-understood algorithm
  • Effective with weak learners (stumps)
  • Built-in feature selection via weak learner choices
  • Theoretical guarantees on training error

Cons

  • Sensitive to noisy data and outliers
  • Sequential training (cannot parallelize)
  • Can overfit with too many rounds on noisy data
  • Outperformed by gradient boosting on most tasks

How It Works

  1. Initialize uniform sample weights $w_i = 1/n$.
  2. Fit weak learner $h_t$ on weighted data. Compute the weighted error: relative loss for each sample and total weighted loss $L_t$.
  3. Compute learner weight $\alpha_t = L_t / (1 - L_t)$. Better learners get higher weight.
  4. Update sample weights: Decrease weights for well-predicted samples, increase for poorly-predicted ones.
  5. Repeat for $T$ rounds. Final prediction is the weighted median.

Key Equations

Relative Error (per sample)

$$e_i^{(t)} = \frac{|y_i - h_t(x_i)|}{\max_j |y_j - h_t(x_j)|}$$

Weighted Loss

$$L_t = \sum_{i=1}^n w_i^{(t)} \cdot e_i^{(t)}$$

Learner Confidence

$$\beta_t = \frac{L_t}{1 - L_t}$$

Weight Update

$$w_i^{(t+1)} = w_i^{(t)} \cdot \beta_t^{1 - e_i^{(t)}}$$

Then normalize: $w_i^{(t+1)} \leftarrow w_i^{(t+1)} / \sum_j w_j^{(t+1)}$

Final Prediction

$$\hat{y}(x) = \text{weighted median}\{h_t(x)\}_{t=1}^T \text{ with weights } \log(1/\beta_t)$$

scikit-learn

Python
from sklearn.ensemble import AdaBoostRegressor
from sklearn.tree import DecisionTreeRegressor
from sklearn.metrics import mean_squared_error, r2_score

model = AdaBoostRegressor(
    estimator=DecisionTreeRegressor(max_depth=3),
    n_estimators=50,
    learning_rate=0.1,
    loss='linear',
    random_state=42
)
model.fit(X_train, y_train)

y_pred = model.predict(X_test)
print(f"R-squared: {r2_score(y_test, y_pred):.4f}")
print(f"RMSE:      {mean_squared_error(y_test, y_pred, squared=False):.4f}")
