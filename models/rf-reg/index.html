Overview

Random Forest combines many decision trees trained on different bootstrap samples of the data, each considering a random subset of features at every split. The final prediction is the average of all individual tree predictions. This reduces variance while maintaining the flexibility of tree-based models.

When to Use

  • You need a strong out-of-the-box baseline with minimal tuning
  • Feature importance ranking is needed
  • The dataset has mixed feature types and nonlinear relationships
  • Robustness to outliers and noise is important

Pros

  • Excellent out-of-the-box performance
  • Robust to overfitting compared to single trees
  • Embarrassingly parallel — trees train independently
  • Built-in feature importance via impurity decrease
  • Works well with high-dimensional data

Cons

  • Less interpretable than a single tree
  • Higher memory and compute cost
  • Cannot extrapolate beyond training range
  • May underperform gradient boosting on structured data
  • Slow prediction for very large ensembles

How It Works

The Random Forest algorithm for regression:

  1. Bootstrap: Draw $B$ random samples (with replacement) from the training set.
  2. Build trees: For each bootstrap sample, grow a decision tree. At each node, randomly select $m$ features (typically $m \approx p/3$ for regression) and find the best split among those features only.
  3. Aggregate: Average the predictions from all $B$ trees.

The combination of bagging (bootstrap aggregating) and random feature subsets ensures that individual trees are decorrelated, which reduces the variance of the ensemble without substantially increasing bias.

Key Equations

Ensemble Prediction

$$\hat{f}_{RF}(x) = \frac{1}{B}\sum_{b=1}^{B} T_b(x)$$

where $T_b(x)$ is the prediction of the $b$-th tree.

Variance Reduction via Averaging

$$\text{Var}\!\left(\frac{1}{B}\sum_{b=1}^{B}T_b\right) = \rho\sigma^2 + \frac{1-\rho}{B}\sigma^2$$

where $\rho$ is the average pairwise correlation between trees and $\sigma^2$ is the variance of a single tree. Random feature subsets reduce $\rho$, lowering ensemble variance.

Out-of-Bag (OOB) Error

$$\text{OOB-MSE} = \frac{1}{n}\sum_{i=1}^{n}\left(y_i - \hat{f}_{OOB}(x_i)\right)^2$$

where $\hat{f}_{OOB}(x_i)$ averages only trees for which $x_i$ was NOT in the bootstrap sample.

Feature Importance (Mean Decrease in Impurity)

$$\text{Imp}(j) = \frac{1}{B}\sum_{b=1}^{B}\sum_{t \in T_b} \Delta_t \cdot \mathbf{1}(\text{split on } j)$$

scikit-learn

Python
from sklearn.ensemble import RandomForestRegressor
from sklearn.model_selection import train_test_split
from sklearn.metrics import mean_squared_error, r2_score

# Split data
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# Fit model
model = RandomForestRegressor(
    n_estimators=100,
    max_depth=None,
    min_samples_split=5,
    max_features='sqrt',
    oob_score=True,
    n_jobs=-1,
    random_state=42
)
model.fit(X_train, y_train)

# Predict
y_pred = model.predict(X_test)

# Evaluate
print(f"R-squared:  {r2_score(y_test, y_pred):.4f}")
print(f"RMSE:       {mean_squared_error(y_test, y_pred, squared=False):.4f}")
print(f"OOB Score:  {model.oob_score_:.4f}")

# Feature importance
for name, imp in sorted(zip(feature_names, model.feature_importances_),
                         key=lambda x: -x[1]):
    print(f"  {name}: {imp:.4f}")
